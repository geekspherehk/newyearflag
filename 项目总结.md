# 年度Flag管理工具 - 项目总结

## 🎯 项目概述

这是一个功能完整的个人年度flag管理工具，帮助用户设定新年目标、评估可行性并跟踪进度。工具提供了多种使用方式，包括命令行界面、图形界面和自动化提醒功能。

## 📁 项目文件结构

```
NewYearFlag/
├── main.py                    # 核心功能模块（Flag管理器）
├── cli.py                     # 命令行界面
├── web_app.py                 # Web应用（Flask）
├── check_reminder.py          # 进度检查提醒脚本
├── launcher.py                # 启动器菜单
├── create_sample_data.py      # 示例数据生成器
├── README.md                  # 使用说明文档
└── flags.json                 # 数据文件（自动生成）
```

## 🚀 功能特点

### 1. 核心功能
- ✅ **Flag管理**: 添加、编辑、删除、查看个人年度目标
- ✅ **可行性评估**: 智能评估flag的可行性（0-100分）
- ✅ **进度跟踪**: 实时更新和查看完成进度
- ✅ **分类管理**: 支持自定义分类（学习、健康、职业等）
- ✅ **数据持久化**: 本地JSON文件存储，支持备份和迁移

### 2. 智能评估
- 📅 **时间合理性**: 评估目标时间的合理性
- 📝 **描述具体性**: 检查描述的具体程度和可执行性
- 📊 **可量化指标**: 识别是否包含量化目标
- 🎯 **目标明确性**: 评估目标的清晰度和可操作性

### 3. 提醒功能
- 🔔 **月度提醒**: 自动提醒长时间未检查的flags
- ⏰ **到期提醒**: 提醒即将到期的目标
- 📈 **进度检查**: 定期生成进度报告
- 💡 **智能建议**: 提供改进建议和完成策略

### 4. 用户界面
- 💻 **命令行界面**: 功能完整，适合技术用户
- 🖥️ **图形界面**: 基于tkinter，操作简单直观
- 📋 **启动菜单**: 统一入口，方便选择使用方式

## 🎯 使用场景

### 1. 新年目标设定
- 在年初设定年度flag
- 获得可行性评估和建议
- 制定具体的执行计划

### 2. 日常进度跟踪
- 定期更新目标进度
- 查看完成情况和趋势
- 调整执行策略

### 3. 月度回顾
- 检查长时间未更新的目标
- 识别需要关注的风险点
- 制定下月行动计划

### 4. 年终总结
- 生成年终进度报告
- 分析完成情况和经验教训
- 为下一年制定更好的目标

## 📈 使用建议

### 1. 目标设定
- 参考可行性评估结果
- 确保目标具体可量化
- 设置合理的时间范围

### 2. 进度跟踪
- 每月至少更新一次进度
- 记录具体的完成情况
- 及时调整执行策略

### 3. 分类管理
- 使用分类功能组织目标
- 平衡不同类型目标
- 关注重要但不紧急的目标

### 4. 定期回顾
- 关注月度提醒
- 分析完成率低的原因
- 学习和改进目标设定方法
